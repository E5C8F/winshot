WinShot

轻量级 Windows 截图库

基于 Windows API 的纯 Python 截图解决方案，支持多种截图方式、后台截图、DPI 感知，无需任何第三方依赖。

纯 Python 实现 零依赖 支持后台截图 DPI 感知

功能特性

📸

多种截图方式

支持 BitBlt、StretchBlt、PrintWindow 三种截图方式，满足不同场景需求。

👻

后台截图

支持对被遮挡或最小化的窗口进行截图，无需将窗口置于前台。

📐

区域截图

支持指定坐标和尺寸进行区域截图，灵活获取所需内容。

🔄

图像缩放

StretchBlt 方式支持截图时直接缩放到指定尺寸，高效便捷。

🖥️

DPI 感知

内置 DPI 感知设置，完美适配高分辨率显示器。

💾

BMP 保存

支持将截图数据直接保存为 BMP 格式图片文件。

API 文档

核心截图函数

BitBlt截图(x1, y1, cx, cy, hwnd)
别名: bitblt_screenshot
通过 BitBlt 对指定窗口进行后台截图，支持区域截图。

参数

  • x1, y1 int - 源矩形左上角的坐标（相对于窗口左上角）
  • cx, cy int - 要截取的区域宽高（像素）
  • hwnd HWND - 窗口句柄，默认为桌面窗口句柄

返回值

返回二进制格式的 BGRA 图像数据（bytearray），包含 width 和 height 属性。

StretchBlt截图(wDest, hDest, xSrc, ySrc, wSrc, hSrc, hwnd)
别名: stretchblt_screenshot
通过 StretchBlt 对指定窗口进行后台截图并缩放到目标尺寸。

参数

  • wDest, hDest int - 目标矩形的宽高（缩放后的尺寸）
  • xSrc, ySrc int - 源矩形左上角的坐标
  • wSrc, hSrc int - 源矩形的宽高
  • hwnd HWND - 窗口句柄，默认为桌面窗口句柄

返回值

返回二进制格式的 BGRA 图像数据（bytearray），包含 width 和 height 属性。

PrintWindow截图(w, h, hwnd)
别名: printwindow_screenshot
通过 PrintWindow API 对窗口进行后台截图，适用于被遮挡的窗口。

参数

  • w, h int - 目标窗口的宽高
  • hwnd HWND - 窗口句柄

返回值

返回二进制格式的 BGRA 图像数据（bytearray），包含 width 和 height 属性。

便捷函数

全屏截图()
别名: full_screenshot
获取整个屏幕的截图。

返回值

返回二进制格式的 BGRA 图像数据。

前台窗口截图(hwnd)
别名: foreground_window_screenshot
获取指定窗口的截图。注意：如果窗口被遮挡，截图结果也会包含遮挡内容。

参数

  • hwnd HWND - 窗口句柄，默认为桌面窗口句柄

返回值

返回二进制格式的 BGRA 图像数据。

辅助函数

获取窗口位置(hwnd)
别名: get_window_position
获取窗口的位置坐标。

参数

  • hwnd HWND - 窗口句柄，默认为桌面窗口句柄

返回值

返回元组 (left, top, right, bottom)，表示窗口的坐标位置。

设置DPI感知(PROCESS_DPI_AWARENESS)
别名: set_process_dpi_awareness
设置进程的 DPI 感知级别，建议在程序启动时调用。

参数

  • PROCESS_DPI_AWARENESS int - DPI 感知级别：0=不感知, 1=系统DPI感知, 2=按监视器DPI感知（默认推荐）

返回值

返回 0 (S_OK) 表示成功，负数表示失败。

保存图片(bitmap_data, filepath, width, height)
别名: save_image
将位图数据保存为 BMP 格式图片文件。

参数

  • bitmap_data bytes - 二进制格式的 BGRA 图像数据
  • filepath str - 图片文件保存路径
  • width, height int - 图像宽高（可选）

使用示例

示例 1: 基础截图
import 截图

截图.设置DPI感知()
data = 截图.全屏截图()
截图.保存图片(data, "screenshot.bmp")
示例 2: 区域截图
import 截图

data = 截图.BitBlt截图(100, 100, 800, 600)
截图.保存图片(data, "region.bmp")
示例 3: 窗口截图
import 截图
import ctypes

user32 = ctypes.windll.user32
hwnd = user32.FindWindowW("Notepad", None)

if hwnd:
    left, top, right, bottom = 截图.获取窗口位置(hwnd)
    data = 截图.PrintWindow截图(right-left, bottom-top, hwnd)
    截图.保存图片(data, "window.bmp")
示例 4: 缩放截图
import 截图

data = 截图.StretchBlt截图(640, 360, 0, 0, 1920, 1080)
截图.保存图片(data, "scaled.bmp")
